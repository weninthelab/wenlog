**Markdown** is a lightweight markup language that helps you format text easily using plain syntax. It's widely used in documentation, blogging platforms, and code repositories.

---

# Headings

Use `#` to create headings:

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

# Text Formatting

```markdown
*Italic* or _Italic_  
**Bold** or __Bold__  
***Bold Italic***  
~~Strikethrough~~
```
*Italic* or _Italic_  
**Bold** or __Bold__  
***Bold Italic***  
~~Strikethrough~~


# Lists

Unordered list:

```markdown
- Item 1
- Item 2
  - Subitem 2.1
```

- Item 1
- Item 2
  - Subitem 2.1


Ordered list:

```markdown
1. First
2. Second
   1. Substep
```

1. First
2. Second
   1. Substep
